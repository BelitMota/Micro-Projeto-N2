# Micro-Projeto-N2

**Objetivo:**
Aplicar na prática os conceitos trabalhados em aula utilizando Git e GitHub,
incluindo versionamento, branches, commits, issues e pull requests.

**Descrição da Atividade**
• 2 alunos com projetos distintos. (HTML/CSS/JavaScript, etc).
• Podem ser mais de 2, até 4 alunos.
• Clonar o projeto do amigo, efetuar alteração no projeto do amigo,
• Efetuar um Pull-Request onde espera-se que o amigo tenha ciência,
aprove ou reprove (preferencia aprovar)
• O repositório no GitHub vocês já devem ter para armazenar o projeto.
• Realizar commits organizados durante o desenvolvimento do projeto.
• Podem ser criados novos branches para desenvolvimento de
funcionalidades ou alterações.
• Podem ser criados novos issues no repositório descrevendo uma tarefa
ou melhoria.
• Resolver a issue e mencionar o número dela no commit realizado.
• Abrir uma Pull Request para realizar o merge das alterações feitas na
branch criada.

# Projeto

Basicamente o projeto sera calculadora básica para identificar o Índice de Massa Corporal (IMC). Utilizando a fórmula:
IMC = peso / altura^2

Nele sera incluido:
- html
- css
- logica python

Saidas esperadas:

    Faixa de IMC            Classificação
    Abaixo de 18,5          Abaixo do peso
    Entre 18,5 e 24,9       Peso normal
    Entre 25,0 e 29,9       Sobrepeso
    Entre 30,0 e 34,9       Obesidade Grau I
    Entre 35,0 e 39,9       Obesidade Grau II (severa)
    Acima de 40,0           Obesidade Grau III (morbida)
